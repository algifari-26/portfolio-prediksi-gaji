# Salary Prediction Model (Indonesia Case Study)

### Tentang Proyek
Proyek ini adalah simulasi **Data Mining** untuk memprediksi standar gaji karyawan di Indonesia berdasarkan tahun pengalaman kerja. Tujuannya adalah membantu HRD atau pelamar kerja mengetahui estimasi gaji yang wajar menggunakan pendekatan statistik.

### Tools & Teknologi
* **Bahasa:** Python
* **Library:** Pandas, Scikit-Learn, NumPy
* **Algoritma:** Simple Linear Regression
* **Environment:** Google Colab / Jupyter Notebook

### Hasil Evaluasi
* Model berhasil mempelajari pola "Linear Positive Relationship" antara pengalaman dan gaji.
* Prediksi Gaji untuk Pengalaman 5 Tahun: **Rp 14-15 Juta** (Estimasi).
